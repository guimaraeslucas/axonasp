Production-ready deployment

Deploy AxonASP in minutes, scale with confidence.

AxonASP supports fast container-based startup, native cross-platform builds, and two infrastructure topologies: reverse proxy mode and FastCGI mode.

Default proxy port: 8801 FastCGI endpoint: 9000 CLI execution support
Download binaries Open repository

Deployment principles

  • Serve static content directly in your edge web server.
  • Forward dynamic ASP requests to AxonASP runtime.
  • Keep configuration centralized in axonasp.toml and environment values.

Quick start options

Docker

Use the repository Dockerfile and docker-compose to start a full runtime quickly.

git clone https://github.com/guimaraeslucas/axonasp.git
cd axonasp
docker-compose up -d

Native build

Build locally with Go 1.26+ using scripts for Windows, Linux, and macOS.

# Windows
./build.ps1 -Platform "windows" -Architecture "amd64"

# Linux/macOS
./build.sh --platform "linux" --architecture "amd64"

Architecture choices

Reverse proxy mode (HTTP runtime)

Run axonasp-http behind Nginx, Apache, Caddy, or IIS for TLS termination and static asset caching.

FastCGI mode

Route ASP execution through FastCGI directly from your web server for integrated request flow.

CLI mode

Execute ASP, VBS, and JavaScript scripts in operational tasks, background jobs, and automation pipelines.

Nginx and Apache examples

Nginx reverse proxy

server {
    listen 80;
    server_name myapp.local;

    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        root /var/www/axonasp/www;
    }

    location / {
        proxy_pass http://127.0.0.1:8801;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}

Nginx FastCGI

server {
    listen 80;
    server_name myapp.local;
    root /var/www/axonasp/www;

    location ~ \.asp$ {
        include fastcgi_params;
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    }
}

Apache reverse proxy

<VirtualHost *:80>
    ServerName myapp.local
    DocumentRoot "/var/www/axonasp/www"

    ProxyPassMatch "^/(.*\.jpg|png|css|js)$" "!"
    ProxyPass / http://127.0.0.1:8801/
    ProxyPassReverse / http://127.0.0.1:8801/
</VirtualHost>

Apache FastCGI

<VirtualHost *:80>
    ServerName myapp.local
    DocumentRoot "/var/www/axonasp/www"

    <FilesMatch "\.asp$">
        SetHandler "proxy:fcgi://127.0.0.1:9000"
    </FilesMatch>
</VirtualHost>

Build-time optimization tags

Disable selected libraries when needed using tags such as lib_g3image_disabled and lib_g3pdf_disabled to produce leaner binaries. 
# Windows
./build.ps1 -Tags "lib_adodb_disabled lib_msxml_disabled"

# Linux/macOS
./build.sh --tags "lib_g3image_disabled,lib_g3pdf_disabled"
