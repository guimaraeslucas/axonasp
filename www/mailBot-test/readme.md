<h4>mailBot</h4>
<p>We all face situations where we needed to find and isolate email addresses. mailBot searches for email addresses in Access databases, Excel sheets, text files and html documents.</p>
<p>Key features:</p><ul>
<li>mailBot isolates 1000's of email addresses from any text, Access databases, Excel sheets, Text files and HTML files</li>
<li>mailBot lists each email address only once</li>
<li>mailBot is very easy to use</li>
<li>mailBot does not change files and it does not store anything in a database</li>
<li>mailBot is secured by a password, to avoid exposure of sensitive data</li>
<li>mailBot is very fast - it finds up to 1000 email addresses/second</li>
<li>mailBot deals with very big Access databases (up to 500 MB)</li>
</ul>
<h4>Hosting requirements</h4>
<p>mailBot runs on any Windows XP Professional, Vista, or Server Edition.</p>
<h4>Installation</h4>
<p>Upload the archive to your host. Make sure to upload the files to search to the "files" folder. The "files" folder needs read/write-permissions for IUSR.</p>
