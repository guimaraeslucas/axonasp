# Methods

## Overview

This page lists methods exposed by G3FC.

## Method List
- Create: Creates a new resource for subsequent operations.
- Extract: Extracts archived content into output paths.
- ExtractSingle: Extracts Single from an archive into output paths.
- Find: Finds matching items using current search criteria.
- Info: Returns structured runtime information for the current context.
- List: Returns structured runtime information for the current context.

## Remarks

- Method names are case-insensitive.
- Validate input types and return values in production code.
- Use Set when assigning object return values.
