# Methods

## Overview

This page lists methods exposed by G3IMAGE.

## Method List
- Clear: Clears the current operation context.
- Close: Closes the current resource and releases handles.
- DrawCircle: Draws Circle onto the current image canvas.
- DrawEllipse: Draws Ellipse onto the current image canvas.
- DrawImage: Draws Image onto the current image canvas.
- DrawLine: Draws Line onto the current image canvas.
- DrawRectangle: Draws Rectangle onto the current image canvas.
- DrawString: Draws String onto the current image canvas.
- DrawStringAnchored: Draws String Anchored onto the current image canvas.
- Fill: Executes the Fill operation provided by the G3IMAGE library.
- LoadFontFace: Loads Font Face into the current operation context.
- LoadImage: Loads Image into the current operation context.
- LoadJPG: Loads JPG into the current operation context.
- LoadPNG: Loads PNG into the current operation context.
- MeasureString: Measures String and returns dimension metrics.
- New: Executes the New operation provided by the G3IMAGE library.
- RenderViaTemp: Renders Via Temp from the current operation context.
- SaveJPG: Saves JPG from the current operation context.
- SavePNG: Saves PNG from the current operation context.
- SetColor: Sets Color for the G3IMAGE library.
- SetHexColor: Sets Hex Color for the G3IMAGE library.
- SetLineWidth: Sets Line Width for the G3IMAGE library.
- Stroke: Executes the Stroke operation provided by the G3IMAGE library.

## Remarks

- Method names are case-insensitive.
- Validate input types and return values in production code.
- Use Set when assigning object return values.
