# G3MD Methods

## Overview
This page provides a summary of the methods available in the **G3MD** library for converting Markdown to HTML in AxonASP.

## Method List

- **Process**: Converts a Markdown-formatted string into a string of HTML. Returns a **String**.

## Remarks
- The method name is case-insensitive.
- The output HTML follows the GitHub Flavored Markdown (GFM) specification.
- Use the **Process** method after configuring the desired properties like **HardWraps** or **Unsafe**.
