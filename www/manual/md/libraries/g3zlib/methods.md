# Methods

## Overview

This page lists methods exposed by G3ZLIB.

## Method List
- Clear: Clears the current operation context.
- Compress: Compresses data into a compact representation.
- CompressFile: Compresses File into a compact representation.
- CompressMany: Compresses Many into a compact representation.
- Decompress: Decompresses content back to its original representation.
- DecompressFile: Decompresses File back to original content.
- DecompressMany: Decompresses Many back to original content.
- DecompressText: Decompresses Text back to original content.

## Remarks

- Method names are case-insensitive.
- Validate input types and return values in production code.
- Use Set when assigning object return values.
