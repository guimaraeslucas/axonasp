# Methods

## Overview

This page lists methods exposed by G3PDF.

## Method List
- AddPage: Adds Page to the current operation context.
- Cell: Writes a positioned cell into the current PDF page.
- Close: Closes the current resource and releases handles.
- Image: Places an image into the current PDF page.
- Line: Draws a line segment in the current PDF page.
- Ln: Moves the PDF cursor to the next line position.
- MultiCell: Writes wrapped multi-line text blocks in the current PDF page.
- Rect: Draws a rectangle in the current PDF page.
- Reset: Resets the current state to defaults.
- SetAutoPageBreak: Sets Auto Page Break for the G3PDF library.
- SetDrawColor: Sets Draw Color for the G3PDF library.
- SetFillColor: Sets Fill Color for the G3PDF library.
- SetFont: Sets Font for the G3PDF library.
- SetFontSize: Sets Font Size for the G3PDF library.
- SetLeftMargin: Sets Left Margin for the G3PDF library.
- SetLineWidth: Sets Line Width for the G3PDF library.
- SetMargins: Sets Margins for the G3PDF library.
- SetRightMargin: Sets Right Margin for the G3PDF library.
- SetTextColor: Sets Text Color for the G3PDF library.
- SetTopMargin: Sets Top Margin for the G3PDF library.
- SetX: Sets X for the G3PDF library.
- SetXY: Sets XY for the G3PDF library.
- SetY: Sets Y for the G3PDF library.
- Text: Writes text at absolute coordinates in the current PDF page.
- Write: Writes content to the active output target.

## Remarks

- Method names are case-insensitive.
- Validate input types and return values in production code.
- Use Set when assigning object return values.
