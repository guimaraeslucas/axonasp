# axexecute

## Overview
Executes a system command through the shell and captures the combined output in G3Pix AxonASP.

## Syntax
```asp
result = obj.axexecute(command)
```

## Parameters and Arguments
- **command** (String): The command line to be executed by the system shell.

## Return Values
Returns a String containing the combined standard output (stdout) and standard error (stderr) generated by the command.

## Remarks
On Windows systems, the command is executed via `cmd.exe /c`. On Unix-like systems, it uses `sh -c`. This function should be used with extreme caution to avoid security risks, especially when passing user-provided input as part of the command.

## Code Example
```asp
<%
Option Explicit
Dim obj, output
Set obj = Server.CreateObject("G3AXON.FUNCTIONS")

' List files in the current directory (Windows example)
output = obj.axexecute("dir")
Response.Write "<pre>" & output & "</pre>"

Set obj = Nothing
%>
```
