# G3STRINGBUILDER Methods

## Overview
This page summarizes methods available in the G3STRINGBUILDER library for efficient string construction in G3Pix AxonASP.

## Methods Reference

| Method | Returns | Description |
|---|---|---|
| Append | Empty | Appends one string value to the internal builder buffer. |
| ToString | String | Returns the current accumulated text as one string. |

## Remarks
- Method names are case-insensitive.
- Append changes internal state.
- ToString is read-only and does not clear the current buffer.
