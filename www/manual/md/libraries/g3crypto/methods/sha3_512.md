# SHA3_512 Method

## Overview
Computes a cryptographic hash using the SHA3-512 (Keccak-512) operation.

## Syntax
```asp
result = crypto.SHA3_512(input)
```

## Parameters and Arguments
- **input** (String or Array, Required): The text or binary byte array to be hashed.

## Return Values
Returns a **String** containing the SHA3-512 hash in lowercase hexadecimal format (128 characters).

## Remarks
SHA3-512 provides the highest level of security in the SHA-3 family, suitable for ultra-secure data integrity and digital signature applications.

## Code Example
```asp
<%
Dim crypto, hash
Set crypto = Server.CreateObject("G3CRYPTO")
hash = crypto.SHA3_512("Ultra Secure Payload")
Response.Write "SHA3-512: " & hash
Set crypto = Nothing
%>
```
