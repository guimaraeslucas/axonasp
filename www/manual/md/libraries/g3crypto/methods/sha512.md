# SHA512 Method

## Overview
Computes a cryptographic hash using the SHA-512 (Secure Hash Algorithm 2, 512-bit) operation.

## Syntax
```asp
result = crypto.SHA512(input)
```

## Parameters and Arguments
- **input** (String or Array, Required): The text or binary byte array to be hashed.

## Return Values
Returns a **String** containing the SHA-512 hash in lowercase hexadecimal format (128 characters).

## Remarks
SHA-512 is the most robust algorithm in the SHA-2 family, providing excellent collision resistance for critical cryptographic tasks.

## Code Example
```asp
<%
Dim crypto, hash
Set crypto = Server.CreateObject("G3CRYPTO")
hash = crypto.SHA512("Maximum Security Hash")
Response.Write "SHA-512: " & hash
Set crypto = Nothing
%>
```
