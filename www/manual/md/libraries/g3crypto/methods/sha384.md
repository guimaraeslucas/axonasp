# SHA384 Method

## Overview
Computes a cryptographic hash using the SHA-384 (Secure Hash Algorithm 2, 384-bit) operation.

## Syntax
```asp
result = crypto.SHA384(input)
```

## Parameters and Arguments
- **input** (String or Array, Required): The text or binary byte array to be hashed.

## Return Values
Returns a **String** containing the SHA-384 hash in lowercase hexadecimal format (96 characters).

## Remarks
SHA-384 is part of the SHA-2 family and is often used in high-security environments where a larger digest than SHA-256 is required.

## Code Example
```asp
<%
Dim crypto, hash
Set crypto = Server.CreateObject("G3CRYPTO")
hash = crypto.SHA384("Secure Data Transmission")
Response.Write "SHA-384: " & hash
Set crypto = Nothing
%>
```
