# VerifyPassword Method

## Overview
Validates a plain-text password against a stored bcrypt hash.

## Syntax
```asp
isValid = crypto.VerifyPassword(password, hash)
```

## Parameters and Arguments
- **password** (String, Required): The plain-text password provided by the user.
- **hash** (String, Required): The bcrypt hash previously generated by the **HashPassword** method.

## Return Values
Returns a **Boolean** value. It returns **True** if the password matches the hash, and **False** otherwise.

## Remarks
This method is the standard way to authenticate users in AxonASP applications. It internally handles the salt extraction and computational cost validation required by the bcrypt algorithm.

## Code Example
```asp
<%
Dim crypto, storedHash, userPass
Set crypto = Server.CreateObject("G3CRYPTO")

' Assume storedHash was retrieved from a database
storedHash = "$2a$10$..." 
userPass = Request.Form("password")

If crypto.VerifyPassword(userPass, storedHash) Then
    Response.Write "Login Success"
Else
    Response.Write "Invalid Credentials"
End If

Set crypto = Nothing
%>
```
