# SHA256 Method

## Overview
Computes a cryptographic hash using the SHA-256 (Secure Hash Algorithm 2, 256-bit) operation.

## Syntax
```asp
result = crypto.SHA256(input)
```

## Parameters and Arguments
- **input** (String or Array, Required): The text or binary byte array to be hashed.

## Return Values
Returns a **String** containing the SHA-256 hash in lowercase hexadecimal format (64 characters).

## Remarks
SHA-256 is an industry-standard hashing algorithm suitable for data integrity checks and modern cryptographic operations. The G3CRYPTO implementation is optimized for performance and is binary-safe for both string and byte array inputs.

## Code Example
```asp
<%
Dim crypto, hash
Set crypto = Server.CreateObject("G3CRYPTO")
hash = crypto.SHA256("Hello, AxonASP!")
Response.Write "SHA-256: " & hash
Set crypto = Nothing
%>
```
