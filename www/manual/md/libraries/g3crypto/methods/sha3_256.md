# SHA3_256 Method

## Overview
Computes a cryptographic hash using the SHA3-256 (Keccak-256) operation.

## Syntax
```asp
result = crypto.SHA3_256(input)
```

## Parameters and Arguments
- **input** (String or Array, Required): The text or binary byte array to be hashed.

## Return Values
Returns a **String** containing the SHA3-256 hash in lowercase hexadecimal format (64 characters).

## Remarks
SHA-3 is the latest generation of the Secure Hash Algorithm standard. It provides high security and resistance against collision attacks. The G3CRYPTO implementation follows the official NIST FIPS 202 standard.

## Code Example
```asp
<%
Dim crypto, hash
Set crypto = Server.CreateObject("G3CRYPTO")
hash = crypto.SHA3_256("AxonASP Secure Data")
Response.Write "SHA3-256: " & hash
Set crypto = Nothing
%>
```
