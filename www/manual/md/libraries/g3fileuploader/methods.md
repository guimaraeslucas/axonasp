# Methods

## Overview

This page lists methods exposed by G3FILEUPLOADER.

## Method List
- AllowExtension: Allows Extension in the current validation rules.
- AllowExtensions: Allows Extensions in the current validation rules.
- BlockExtension: Blocks Extension in the current validation rules.
- BlockExtensions: Blocks Extensions in the current validation rules.
- Form: Reads parsed form fields from the current upload/request context.
- GetAllFilesInfo: Gets All Files Info from the G3FILEUPLOADER library.
- GetFileInfo: Gets File Info from the G3FILEUPLOADER library.
- IsValidExtension: Checks whether Valid Extension is valid or enabled.
- Process: Processes current inputs using current configuration.
- ProcessAll: Processes All using current inputs and configuration.
- SetUseAllowedOnly: Sets Use Allowed Only for the G3FILEUPLOADER library.

## Remarks

- Method names are case-insensitive.
- Validate input types and return values in production code.
- Use Set when assigning object return values.
