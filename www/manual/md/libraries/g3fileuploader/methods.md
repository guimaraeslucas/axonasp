# G3FILEUPLOADER Methods

## Overview
The `G3FILEUPLOADER` object provides methods to govern upload permissions, extract single items, process uploads automatically, and explore form variables.

## Available Methods

* `AllowExtension`
* `AllowExtensions`
* `BlockExtension`
* `BlockExtensions`
* `Form`
* `GetAllFilesInfo`
* `GetFileInfo`
* `IsValidExtension`
* `Process`
* `ProcessAll`
* `SetUseAllowedOnly`
