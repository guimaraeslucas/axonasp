# Security Policy

## Supported Versions

Only the current main version of the repository is supported with security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within AxonASP, please do NOT open a public issue. 
Instead, please send an e-mail to Lucas Guimarães at [guimaraes.lucas@gmail.com](mailto:guimaraes.lucas@gmail.com).

All security vulnerabilities will be promptly addressed.
